# North Web Starter

一个基于 React、TypeScript 和 Vite 的轻量 Web 项目骨架，包含响应式首页、基础组件拆分与完整构建配置。

## 开始使用

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

## 目录结构

```text
src/
├── components/     # 可复用组件
├── styles/         # 全局样式与设计变量
├── App.tsx         # 页面结构
└── main.tsx        # 应用入口
```
